Exemplos de código

Comece em minutos com nossas APIs de inferência.

cURL 
curl -X POST https://api.nova-ia.dev/v1/chat/completions \
  -H "Authorization: Bearer $NOVA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-large",
    "messages": [{"role": "user", "content": "Explique RAG em 2 frases."}]
  }'
JavaScript 
const res = await fetch('https://api.nova-ia.dev/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.NOVA_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'gpt-large',
    messages: [{ role: 'user', content: 'Crie 3 ideias de app de IA.' }]
  })
});
const json = await res.json();
console.log(json.choices[0].message.content);
Python 
import os, requests

api_key = os.environ.get('NOVA_API_KEY')
payload = {
  'model': 'gpt-large',
  'messages': [ {'role': 'user', 'content': 'Resuma IA generativa.'} ]
}
r = requests.post('https://api.nova-ia.dev/v1/chat/completions',
                  headers={'Authorization': f'Bearer {api_key}'},
                  json=payload)
print(r.json()['choices'][0]['message']['content'])
